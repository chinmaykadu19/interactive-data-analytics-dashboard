# 📊 Interactive Data Analytics Dashboard

## Live Demo
https://data-analytics-dashboard1.streamlit.app/

---

## Overview

This project is an interactive data analytics dashboard built using Python, Pandas, Streamlit, and Plotly. It allows users to upload any CSV dataset and perform dynamic analysis and visualization.

The dashboard automatically detects numeric columns and generates interactive charts for data exploration.

---

## Features

- Upload any CSV dataset
- Automatic dataset preview
- Dataset statistics and metrics
- Interactive visualizations
- Histogram, Scatter Plot, and Line Chart support
- Works with large datasets
- Live dashboard deployment support

---

## Tech Stack

- Python
- Pandas
- Streamlit
- Plotly
